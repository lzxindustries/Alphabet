## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* Qt 5.14.2
* Qt Installer Framework 3.2.2
* CMake 3.16

### Installation

1. Clone
```sh
git clone https://github.com/lzxindustries/lzxtempl-desktop-app.git
```

2. Build
```
cd lzxtempl-desktop-app
mkdir build
cd build
cmake --build .
```
